**Installation**
1. run "git clone https://github.com/jiahow99/AdvisoryApp-CRUD.git"
2. run "composer install"
3. run "cp .env.example .env"
4. run "php artisan key:generate"
5. run "php artisan migrate"
6. run "npm install"


**Local Dev**
1. run "php artisan serve"
2. run "npm run dev"
